# 🎟 Ticket Booking System

## Overview

A full-stack Ticket Booking System built using FastAPI and Next.js.

---

## Tech Stack

### Backend

- FastAPI
- PostgreSQL
- SQLAlchemy
- Alembic
- JWT Authentication

### Frontend

- Next.js
- React
- TypeScript
- React Query
- Tailwind CSS
- ShadCN UI

---

## Features

- JWT Authentication
- Venue Management
- Event Management
- Automatic Seat Generation
- Seat Selection
- Prevent Double Booking
- Booking Confirmation
- Ticket Generation
- QR Code Tickets
- PDF Ticket Download
- Cancel Booking
- Dashboard
- My Tickets
- Profile Page

---

## Backend

```bash
cd backend

pip install -r requirements.txt

uvicorn app.main:app --reload
```

---

## Frontend

```bash
cd frontend

npm install

npm run dev
```
